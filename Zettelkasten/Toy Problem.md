# Toy Problem vs Real-World Problem

|                 | Toy                                                                                          | Real-World                                                                        |
| --------------- | -------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **Description** | can be given a concise and exact description                                                 | Tend to not have a single agreed-upon description, but we can give a general idea |
| **Importance**  | To compare performance of different algos                                                    | the solutions people actually care about                                          |
| **Examples**    | [[Sliding-Block Puzzles]], [[n-queens problem]], 4-water Jug problem, River-crossing problem | route-finding, touring, robot navigation, protein folding                         |
