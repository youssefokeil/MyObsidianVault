# Supabase
- Free open source software alternative for Firebase and Amplify
- Firebase & Amplify lock you in to proprietary on a specific cloud platform
- Provides Backend 
	- Database
	- Edge functions
	- file storage
- provides Frontend SDKs
- Very simple row-level security
- Uses [[postgreSQL]]
