# Convolution

$$
f(t)*g(t)=\int_{-\infty}^{\infty} f(\tau).g(t-\tau)
$$
But can be simplified to this if the function is only defined in the positive domain
$$
f(t)*g(t)=\int_{0}^{t} f(\tau).g(t-\tau)
$$
