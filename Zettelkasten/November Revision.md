## 1) Factorizing by Greatest Common Factor GCF
### Steps:
1. Find GCF for numerical factor **(numbers)**
	Ex: $36ab - 24a^{2}b-18ab^{2}$
	Numerical Factors: 36, 24, 18
	
| number | divisor |     | number | divisor |     | number | divisor |
| ------ | ------- | --- | ------ | ------- | --- | ------ | ------- |
| 36     | ==2==   |     | 24     | ==2==   |     | 18     | ==2==   |
| 18     | 2       |     | 12     | 2       |     | 9      | 3       |
| 9      | 3       |     | 6      | 2       |     | 3      | ==3==   |
| 3      | ==3==   |     | 3      | ==3==   |     | 1      |         |
| 1      |         |     | 1      |         |     |        |         |
Greatest Common Factor Numerically: $2*3=6$
--> ==6==

--------------

2. Take smallest power of each variable **(algebraic)**
	Ex: $36ab-24a^{2}b-18ab^{2}$
	smallest powers are --> ==ab==

-------
3. *Multiply* the **number** by the **algebraic factors**
	Ex: $6 * ab$ --> ==6ab==
