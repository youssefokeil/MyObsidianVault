# Image Datasets
### MNIST
- Created 1994
 - 10 classes: 0-9 digits
 - 28x28 grayscale images
 - 50k training images
 - 10k test images
### CIFAR-10
- 10 classes
- 32x32 RGB
- 50k train (5k/class)
- 10k test   (100/class)
### CIFAR-100
- 100 classes
- 32x32 RGB
- 50k train (500/class)
- 10k test   (100/class)
### ImageNet
- Fei-Fei Li prepared by using crowdsourcing on Amazon Mechanical Turk
- 1000 classes
- 256x256 RGB (resized by practitioners)
- 1.3m train
- 50k valid
- 100k test
- {used top-5}
### MIT Places
- Scene Recognition
- 365 classes
- 8m train
- 256x256 RGB (resized)
- 18.25k train
- 328.5k test
### Omniglot
- 1623 classes from 50 different alphabet
- 20 images per category
