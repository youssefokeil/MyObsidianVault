# JUnit

![[Pasted image 20251112164537.png]]
## Workflow of JUnit:
1. ==**Create Test Cases**==
	1. **Test Input:** Any required input data
	2. **Expected Output:** What expected result
	3. **Initialization**: Steps before test execution
		Uses `@BeforeEach` for initialization before each test
	4. **Test Steps:** Interactions (such as method calls), and output
sons
	5. **Tear Down:** Cleanup after execution
		Uses `@AfterEach` for cleanup after each setup.
2. Review/Rework
3. Baseline
4. Execute Test Cases
