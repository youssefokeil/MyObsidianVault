# Software Testing Taxonomy
> you can classify testing by the following taxonomies
## By Knowledge of the System
1. Grey Box Testing
2. White Box Testing
3. Black Box Testing
## By Object of Testing
1. Function Testing
2. UI Testing
3. Security Testing
4. Performance Testing
5. Usability Testing
6. Localization Testing
7. Compatibility Testing
## By Degree of Automation
1. Manual Testing
2. Automated Testing
3. Semi-automated Testing
## By Degree of Isolation of Components
1. **[[User Acceptance Testing]]**
2. **[[System Testing]]**
3. **[[Integration Testing]]**
4. **[[Unit Testing|Unit/Component Testing]]**
## By Time of Testing
1. Regression Testing
2. Smoke Testing
3. Alpha Testing
4. Beta Testing
5. New Feature Testing
## By Degree of Preparation to Be Tested
1. Positive Testing
2. Negative Testing
## By Basis of Positive Scenario
1. Ad Hoc Testing
2. Formal Testing
